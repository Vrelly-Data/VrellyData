## 🎨 Gender Filter Implementation

### Database Storage
Gender values are stored as single characters:
- `M` for Male
- `F` for Female

### Frontend Conversion
The hook `src/hooks/useFreeDataSearch.ts` (lines 51-57) converts UI values:

```typescript
const convertGender = (gender: string): string => {
  if (gender.toLowerCase() === 'male') return 'M';
  if (gender.toLowerCase() === 'female') return 'F';
  return gender;
};
```

This ensures the UI displays human-readable labels while queries use database values.
